# Roadmap

## Delivered in 0.1.0

- Offline, standard-library-only CLI control plane
- Workspace registry and bounded discovery
- New and existing repository onboarding
- Project command detection and executable repository contracts
- SQLite session, validation, failure, and event memory
- Environment reference broker
- Repetition guard
- Doctor, context, validation, commit, push, and ship workflows
- Managed Git hooks
- Secret filename protection
- Linux, macOS, WSL, Termux, and PowerShell installers

## Planned for 0.2

- MCP server exposing workspace resolution, context, validation, and handoff tools
- JSON output modes for all read operations
- Named agent permission profiles
- Pluggable secret providers for 1Password, Bitwarden, and native keychains
- Deployment recipes with approval gates

## Planned for 0.3

- Local dashboard and Android companion
- Multi-agent task leases and concurrency locks
- Repository health summaries and notifications
- Centralized policy bundles for teams
