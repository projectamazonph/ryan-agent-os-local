# Changelog

## 0.1.0 - 2026-07-14

- Added workspace registry and deterministic resolver.
- Added automatic detection for Node, Python, Rust, Go, Maven, and Gradle repositories.
- Added new-repository scaffolding and existing-repository onboarding.
- Added repository-local agent contracts and structured handoffs.
- Added SQLite sessions, failures, command executions, and validation history.
- Added repository fingerprinting and identical-failure loop protection.
- Added environment profile references, presence checks, and child-process injection.
- Added doctor, context, validation, commit, push, and ship workflows.
- Added protected-branch, dirty-worktree, detached-HEAD, and staged-secret guards.
- Added managed pre-push, post-commit, and post-checkout hooks.
- Added bounded repository scanning and built-in recipes.
